# AnaProgDelSoft - Codice

Questo repository contiene il codice per il corso di Analisi e Programmazione del Software.

## Programmi Inclusi

### 1. PrimoCeck.cpp
**Descrizione:** Programma che chiede un numero intero all'utente e verifica se è un numero primo.

**Come compilare e eseguire:**
```bash
g++ PrimoCeck.cpp -o Primo
./Primo
```

**Esempio di utilizzo:**
- Input: 7
- Output: 7 è un numero primo.

### 2. PiramideNumeri.cpp
**Descrizione:** Programma che chiede un numero intero e stampa una piramide di numeri basata su quel numero.

**Come compilare e eseguire:**
```bash
g++ PiramideNumeri.cpp -o Piramide
./Piramide
```

**Esempio di utilizzo:**
- Input: 3
- Output:
```
  1
 121
12321
```
